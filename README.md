# Eslint Config of cicada

## 项目结构

```
cicada-eslint-config
├─ index.js
├─ package.json
├─ README.md
└─ yarn.lock

```

## Install

#### NPM

```shell
npm i @cicada/eslint-config -D
```

#### YARN

```shell
yarn add @cicada/eslint-config --dev
```

## Usage

```js
{
  "extends": ["@cicada/eslint-config"]
}
```
